WarO_Groovy
===========

An example of the newly christened War-O in Groovy.

To run:

-- install Gradle from http://gradle.org
-- configure numCards, numGames and players in config.groovy
-- type (or script): 

gradle myBuild
cd staging
java -jar warO.jar ../config.groovy
